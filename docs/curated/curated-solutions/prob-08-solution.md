## Solution: Problem 8 - While Loop Basics - Countdown!

```dart
void main() {
  // Initialize your countdown variable here
  int countdown = 5;

  // Start your while loop here
  while (countdown > 0) {
    print(countdown);
    countdown--; // Decrement countdown by 1
  }

  // Print "Blast off!" after the loop
  print("Blast off!");
}
```
