## Solution: Problem 5 - Comparing Values

```dart
void main() {
  // Declare your variables here
  int scoreA = 85;
  int scoreB = 92;
  int apples = 5;
  int oranges = 5;

  // Perform comparisons and print the results
  print("Is scoreA > scoreB? ${scoreA > scoreB}");
  print("Is scoreA <= scoreB? ${scoreA <= scoreB}");
  print("Are apples == oranges? ${apples == oranges}");
  print("Are apples != scoreA? ${apples != scoreA}");
}
```
